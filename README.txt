Для запуска проекта нужно выполнить шаги, описанные ниже
Стек проекта: vue(frontend), django и drf для api(backend), nginx(связь frontend и backend частей)

Для запуска frontend части необходимо:
- перейти в папку frontend
- выполнить команду npm install (должно быть установлено node.js)
- возможно придётся установить необходимую библиотеку командой npm install aos
- выполнить команду npm run build
- выполнить команду npm run preview
после чего у вас запустится frontend часть

Для запуска backend части необходимо:
- перейти в папку backend(открыв новое окно терминала)
- выполнить установку зависимостей(версия python должна быть не ниже 10) с помощью команды: pip install -r requirements.txt
- запустить backend часть с помощью команды: python manage.py runserver

Для запуска nginx(позволяет связать frontend и backend) нужно:
- установить nginx в корневую папку проекта
- изменить настройки nginx(прописаны в nginx.conf)
- запустить nginx

При верно выполненных шагах у вас будет локально развёрнут проект на адресе http://localhost
